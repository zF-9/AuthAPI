# Login API prototyping using Node.js 

Node.js with JSON Web Token (JWT) Authentication & Sequelize

 ## Project Requirements ##
    - Express JS
    - bcryptjs 
    - jsonwebtoken (JWT)
    - Sequelize 
    - MySQL

## Project Dependencies
  ### Sequelize
  ```
  npm install --save sequelize
  ```

## Project setup
```
npm install
```

### Run
```
node server.js
```
